  The circR2Cancer is a manually curated database that provides reliable associations between circRNAs and cancers. The current version of the circR2Cancer, which contains 518 circRNA-cancer associations, 477 circRNA-miRNA interactions and 422 miRNA-cancer relationships. At the same time, we offer users a simple friendly interface, making it convenient to browse, search and download the interactions mentioned above.
