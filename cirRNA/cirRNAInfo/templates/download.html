  Welcome to use circR2Cancer. In order to provide the necessary assistance to the relevant staff, we have stored the relevant information in the database in the form of tables and made it available for everyone to download. Each table contains the following contents:
  • circRNA-cancer association table, which contains circRNA name, the name of disease, species, functional describution,etc.
  • circRNA-miRNA association table, which includes the name of circRNA and miRNA.
  • miRNA-cancer association table which contains miRNAs name, related disease, PMID and functional describution.
  • circRNA information table which involves circRNAs name, methods of circRNA detection, expression pattern, chromosome and region, etc.
  • miRNA information table which contains miRNAs name, accession, symbol, description and gene family, etc.
  • cancer information table which includes the name of disease, DOID definition, Xrefs, subsets, synonyms and relationships.
